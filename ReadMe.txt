Чтобы запустить приложение, достаточно просто закинуть проект в идею
(Скорее всего и другие ИДЕ, тестил и писал на идее)

В приложении 3 типа питомца: кошка, собачка, зайчик
Выбор происходит на 1ом стартовом экране
Там же можно ввести имя для питомца

После начала игры, выбранный питомец будет рандомно перемещаться по полю
Со временем у него будет ухудшаться настроение
(По дефолту каждый час, после 12 часов без кормежки питомец умирает)
Со временем питомец будет расти
(По дефолту каждые 6 часов, максимум 5 раз)
Задний фон будет соотвествовать времени компьютера, на котором запущено приложение

Питомца можно покормить, для этого надо нажать соотвествующую кнопку
Для каждого питомца будет будет размещена его любимая еда посреди поля, после чего
он подойдет к ней и съест, а полоска настроения заполнится

По прошествию 12 часов с момента последней кормежки питомец умрет и следующие 24 часа будет
выводится экран смерти, по прошествию которых можно будет начать новую игру

Сохранения хранятся в текстовом файле sample.resources.save.save.txt
Чтобы насильно начать новую игру достаточно все удалить из файла сохранения
или в классе Main перед Saver.read(); вызвать Saver.clearSaveFile();

Для тестирования в классе Pet есть 2 констаты, отвечающие за период роста и изменения настроения
(GROW_PERIOD и MOOD_PERIOD соответственно. При тестировании ставил их на секунду, т.е. 1000)
Также в классе Saver надо будет поменять константу CHECK_PERIOD, чтобы изменения отображались
(Также ставил ее на секунду)
